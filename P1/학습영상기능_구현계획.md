# 📹 강의 섹션별 영상 학습 구현 계획

## 🎯 목표

특정 강의를 수강신청 후, "학습하기" 클릭 시 아래 구조로 구현:

```
강의 상세 페이지
    ↓
"학습하기" 버튼
    ↓
학습 페이지 (섹션 선택)
    ├─ 왼쪽: 섹션 목록
    │  ├─ 섹션 1
    │  │  ├─ 강의 1-1
    │  │  ├─ 강의 1-2
    │  │  └─ 강의 1-3
    │  ├─ 섹션 2
    │  │  ├─ 강의 2-1
    │  │  └─ 강의 2-2
    │  └─ ...
    └─ 오른쪽: 동영상 재생
       ├─ 비디오 플레이어
       ├─ 재생 컨트롤
       └─ 진행률 저장
```

---

## 📊 데이터 구조

### 현재 엔티티 관계

```
Course (강의)
  ↓ 1:N
Section (섹션/단원)
  ├─ id
  ├─ course_id (FK to Course)
  ├─ title (예: "1단원: 기초 개념")
  └─ sort_order
      ↓ 1:N
      Lecture (개별 강의/영상)
      ├─ id
      ├─ section_id (FK to Section)
      ├─ title (예: "1-1. 주식의 기초")
      ├─ video_url (예: "https://example.com/video.mp4")
      ├─ play_time (영상 길이, 초)
      └─ sort_order

LectureProgress (학습 진행 상황)
├─ id
├─ member_id (사용자)
├─ lecture_id (강의)
├─ last_position (마지막 재생 위치, 초)
└─ updated_at
```

## 📱 FE 페이지 구조

**기능:**

│     └─ 2단원                        │

- 이전/다음 강의 버튼

## 🎬 비디오 플레이어 선택

### 옵션 1: HTML5 Native Video (추천)
```tsx
<video
  ref={videoRef}
  width="100%"
  height="auto"
  controls
  onTimeUpdate={handleTimeUpdate}
  onPlay={handlePlay}
  onPause={handlePause}
>
  <source src={lectureUrl} type="video/mp4" />
  브라우저가 비디오를 지원하지 않습니다.
</video>
```

**장점:**
- 추가 라이브러리 불필요
- 가볍고 빠름
- 대부분 브라우저 지원

**단점:**
- 고급 기능 제한 (조화, 속도 조절 제한 등)

### 옵션 2: Video.js 라이브러리
```tsx
import VideoJS from 'video-react'

<VideoJS
  controls
  autoplay
  preload="auto"
  data={{
    sources: [{ src: lectureUrl, type: 'video/mp4' }],
  }}
/>
```

**장점:**
- 다양한 기능 (속도 조절, 품질 선택 등)
- 스타일링 쉬움

**단점:**
- 추가 의존성
- 번들 크기 증가

## 🔄 사용자 흐름

### 1단계: 강의 선택
```
강의 상세 페이지
  ↓
"지금 수강하기" 버튼 클릭
  ↓
수강신청 완료
```

### 2단계: 학습 시작
```
강의 상세 페이지
  ↓
"학습하기" 버튼 클릭 (또는 "내 강의실" → 강의 선택)
  ↓
GET /api/courses/{courseId}/sections
  ↓
학습 페이지 로드
  ├─ 섹션 목록 표시
  ├─ 첫 강의 자동 로드
  └─ 비디오 재생 준비
```

### 3단계: 강의 시청
```
비디오 재생
  ↓
사용자가 시청
  ↓
[진행 상황 자동 저장 (5초마다)]
  ├─ POST /api/lecture-progress
  └─ { lectureId, lastPosition }
  ↓
다음 강의 버튼 클릭 또는 강의 목록에서 선택
  ↓
다시 반복
```

## 💾 데이터 저장 전략

### 진행 상황 자동 저장

```typescript
// 비디오 재생 위치 업데이트
const handleTimeUpdate = async (currentTime: number) => {
  // 5초마다 저장 (중복 저장 방지)
  if (lastSaveTime && currentTime - lastSaveTime < 5) {
    return;
  }
  
  // API 호출
  await lectureProgressService.saveProgress(lectureId, Math.floor(currentTime));
  
  setLastSaveTime(currentTime);
};

// 페이지 이동 시 저장
useEffect(() => {
  return () => {
    // 컴포넌트 언마운트 시 최종 위치 저장
    if (videoRef.current) {
      lectureProgressService.saveProgress(
        lectureId, 
        Math.floor(videoRef.current.currentTime)
      );
    }
  };
}, [lectureId]);
```

## 🧪 테스트 시나리오

### 1. 정상 플로우
```
1. 강의 상세 페이지에서 "학습하기" 버튼 클릭
2. 학습 페이지 로드
3. 섹션 목록 표시 확인
4. 첫 강의 자동으로 재생
5. 다른 강의 선택 시 변경됨
6. 진행 상황 자동 저장 확인
```

### 2. 토큰 만료
```
1. 학습 중 토큰 만료
2. 진행 상황 저장 요청 → 401 에러
3. 자동으로 로그인 페이지로 리다이렉트
```

### 3. 강의 없음
```
1. 수강하지 않은 강의 ID로 직접 접근
2. 403 에러 또는 "접근 권한 없음" 메시지
```
