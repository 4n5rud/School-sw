## 🎯 구현 목표

사용자가 강의를 수강신청 후 "학습하기"를 클릭하면:
1. 강의 섹션(단원)를 선택할 수 있는 사이드바 表示
2. 각 섹션 내 동영상 강의들을 선택 가능
3. HTML5 비디오 플레이어에서 동영상 재생
4. 시청 진행도 자동 저장

✅ **모든 목표 달성**

### 데이터 구조

```
Course (강의)
  ├─ id: 1
  ├─ title: "주식 초보자를 위한 기초 강좌"
  └─ Section[] (1:N 관계)
       ├─ Section (id=1)
       │  ├─ title: "1단원: 기초 개념"
       │  ├─ sortOrder: 1
       │  └─ Lecture[] (1:N 관계)
       │       ├─ Lecture (id=1)
       │       │  ├─ title: "1-1. 주식의 기초"
       │       │  ├─ videoUrl: "https://..."
       │       │  ├─ playTime: 1200 (초)
       │       │  └─ sortOrder: 1
       │       └─ Lecture (id=2)
       │          ├─ title: "1-2. 차트 분석"
       │          └─ ...
       └─ Section (id=2)
          ├─ title: "2단원: 실전 분석"
          └─ Lecture[]...

LectureProgress (학습 진행 기록)
├─ id: 1
├─ memberId: 1 (수강 학생)
├─ lectureId: 1 (시청 강의)
├─ lastPosition: 600 (마지막 재생 위치, 초)
├─ playTime: 1200 (전체 강의 길이)
└─ watchPercentage: 50 (시청률, %)
```

#### 3. SectionService
```java
@Service
public class SectionService {
    
    /**
     * 강의별 모든 섹션 조회 (강의 포함)
     */
    public List<SectionResponse> getSectionsByCourse(Long courseId) {
        List<Section> sections = sectionRepository.findByCourseId(courseId);
        return sections.stream()
            .sorted(Comparator.comparingInt(Section::getSortOrder))
            .map(SectionResponse::from)
            .collect(Collectors.toList());
    }
    
    /**
     * 섹션별 강의 목록 조회
     */
    public List<LectureResponse> getLecturesBySection(Long sectionId) {
        List<Lecture> lectures = lectureRepository.findBySectionId(sectionId);
        return lectures.stream()
            .sorted(Comparator.comparingInt(Lecture::getSortOrder))
            .map(LectureResponse::from)
            .collect(Collectors.toList());
    }
    
    /**
     * 특정 강의 조회
     */
    public LectureResponse getLectureById(Long lectureId) {
        Lecture lecture = lectureRepository.findById(lectureId)
            .orElseThrow(() -> new IllegalArgumentException("강의를 찾을 수 없습니다"));
        return LectureResponse.from(lecture);
    }
}
```

#### 1. VideoPlayer 컴포넌트

**기능:**
- HTML5 `<video>` 태그 사용
- 시청 진행도 자동 저장 (5초마다)
- 마지막 시청 위치에서 재개
- 시청 비율 진행률 바 표시
- 커스텀 플레이 컨트롤

**주요 코드:**
```typescript
export default function VideoPlayer({
  lectureId,
  videoUrl,
  title,
  playTime,
  onTimeUpdate,
}: VideoPlayerProps) {
  
  // 마지막 시청 위치 로드
  useEffect(() => {
    const progress = await lectureProgressService.getLectureProgress(lectureId);
    if (videoRef.current && progress.lastPosition) {
      videoRef.current.currentTime = progress.lastPosition;
    }
  }, [lectureId]);
  
  // 진행도 자동 저장 (5초마다)
  const handleTimeUpdate = () => {
    if (Math.abs(currentTime - lastSaveTime) >= 5) {
      lectureProgressService.saveLectureProgress(lectureId, Math.floor(currentTime));
      setLastSaveTime(currentTime);
    }
  };
  
  // 페이지 이동 시 최종 위치 저장
  useEffect(() => {
    return () => {
      if (videoRef.current) {
        lectureProgressService.saveLectureProgress(
          lectureId,
          Math.floor(videoRef.current.currentTime)
        );
      }
    };
  }, [lectureId]);
}
```


## 🐛 주의 사항

### 1. 비디오 URL 경로

현재 `videoUrl`은 데이터베이스에서 직접 관리됩니다. 실제 배포 시:
- CDN 연동 필요
- CORS 설정 확인
- 비디오 포맷 확인 (MP4 권장)

### 2. 큰 파일 스트리밍

성능 최적화:
- range request 지원 확인
- 적절한 bitrate 제어
- 버퍼링 시간 모니터링

### 3. 모바일 지원

현재 반응형 디자인 적용됨:
- 태블릿: 2열 레이아웃 유지
- 모바일: 1열 (섹션바 토글 필요 시 추가)
