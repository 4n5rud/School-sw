# StockFlow 프로젝트 - FE(프론트엔드) UI 프로토타입

## 📁 프로젝트 구조

```
fe/
├── app/
│   ├── page.tsx                          # 메인 페이지
│   ├── layout.tsx                        # 루트 레이아웃
│   ├── globals.css                       # 전역 스타일
│   ├── courses/
│   │   ├── page.tsx                      # 강의 목록 & 검색 페이지
│   │   └── [id]/
│   │       └── page.tsx                  # 강의 상세 페이지
│   ├── my-courses/
│   │   └── page.tsx                      # 마이페이지 (내 강의실)
│   └── learning/
│       └── [courseId]/
│           └── [lectureId]/
│               └── page.tsx              # 영상 학습 페이지
├── components/
│   ├── Header.tsx                        # 네비게이션 헤더
│   ├── CourseCard.tsx                    # 강의 카드 컴포넌트
│   └── Footer.tsx                        # 푸터
├── lib/
│   ├── types.ts                          # TypeScript 타입 정의
│   └── mockData.ts                       # Mock 데이터 (더미 강의)
└── package.json
```

## 🎯 구현된 페이지 & 기능

### 1️⃣ **메인 페이지** (`/`)
- 히어로 섹션 (배너)
- 인기 강의 4개 그리드 표시
- 특징 섹션 (3개 특징)

### 2️⃣ **강의 탐색 페이지** (`/courses`)
- 🔍 **검색 기능**: 강의명/강사명으로 검색
- 🏷️ **카테고리 필터**: 전체/주식/암호화폐
- 📊 **강의 카드**: 제목, 강사, 별점, 수강인원, 가격 표시

### 3️⃣ **강의 상세 페이지** (`/courses/[id]`)
- 📖 강의 상세 설명
- 📚 커리큘럼 (섹션 & 강의 단위 표시)
- 💰 가격 및 수강 신청 버튼
- ✨ 강의 특징 라벨

### 4️⃣ **마이페이지** (`/my-courses`)
- 📊 전체 학습 진도 대시보드
- 📚 수강 중인 강의 목록
- ✅ 완강한 강의 목록
- 🎯 개별 강의별 진도율 표시

### 5️⃣ **영상 학습 페이지** (`/learning/[courseId]/[lectureId]`)
- 🎬 비디오 플레이어 (시뮬레이션)
- ⏯️ 플레이/일시정지 제어
- 📍 진도율 표시 바
- 📱 반응형 레이아웃
- 📖 커리큘럼 사이드바
  - 섹션별 강의 목록
  - 이전/다음 강의 네비게이션

## 🎨 기술 스택

- **Framework**: Next.js 16.2.1
- **UI/CSS**: Tailwind CSS v4
- **Language**: TypeScript
- **React**: 19.2.4

## 🔌 백엔드 통합 예정

현재 **Mock 데이터**로 UI 프로토타입을 구현했으며, 다음 단계에서는 백엔드 API로 연결됩니다:

```
API 엔드포인트 예상:
- GET /api/v1/courses          → 강의 목록
- GET /api/v1/courses/{id}     → 강의 상세
- POST /api/v1/enrollments     → 수강 신청
- GET /api/v1/members/me/courses → 마이페이지
- PATCH /api/v1/lectures/{id}/progress → 진도 저장
```
